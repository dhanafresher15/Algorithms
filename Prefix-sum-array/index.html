               Naive Approach of Time Complexity O(k*d) :

              Literally updating the value for the elements in the range :

                          Algorithm :

update_value(arr[], n, q[][], k, val)

 

  for i = 0 to i < k

     a = q[0], b = q[1]

     for j = a to j <= b

        arr[i] += val

 

 

               

                             where, d is the number of array elements in the range. 

                   Efficient Approach of Time Complexity O(n) :

                Using Prefix Sum :

                Algorithm :

update_value(arr[], n, q[][], k, val)

 

  for i = 0 to i < k

   a = q[0], b = q[1]

        arr[a] += val

        if(b+1 < n)

               arr[b+1] -= val

 

        /* apply prefix sum array technique

        to get the resultant array */


        for i = 1 to i < n

        arr[i] += arr[i-1]

 

 

where, n is the number of array elements.
