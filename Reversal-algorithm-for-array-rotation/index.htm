REVERSAL ALGORITHM FOR ARRAY ROTATION

                   Naïve Approach of Time Complexity O(n*k) :

              Literally rotating 3 times by swapping :

                          Algorithm :

rotate(arr[], d, n)

  for i = 0 to i < d

    rotate all elements of arr[] by one ;

               

                             where, n is the number of array elements.

                   Efficient Approach of Time Complexity O(n) :

                Using Reversal Algorithm :

                Algorithm :

rotate(arr[], k, n)

  reverse(arr[], 1, k) ;

  reverse(arr[], k + 1, n);

  reverse(arr[], 1, n);

                  

                             where, n is the number of array elements.
